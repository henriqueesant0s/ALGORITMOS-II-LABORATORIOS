# 📘 Laboratório 01 - Funções

## 📌 Descrição

Este laboratório tem como objetivo introduzir o conceito de **funções em C**, permitindo dividir o programa em partes menores e reutilizáveis.

---

## 🧠 Conceito

Funções são blocos de código que executam tarefas específicas, podendo receber parâmetros e retornar valores.

---

## 📚 Conteúdos abordados

- Declaração de funções
- Passagem de parâmetros
- Retorno de valores
- Organização do código

---

## 🎯 Objetivo do laboratório

Aprender a estruturar programas de forma modular, evitando códigos repetitivos e melhorando a legibilidade.

---

## 💻 Compilação

```bash
gcc main.c -o programa.exe
```

## ▶️ Execução

```bash
./programa.exe
```
